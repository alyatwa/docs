## Get started with Benchwrk

Follow these steps to set up Benchwrk for logging and monitoring your applications.

### Step 1: Create your Benchwrk account

<AccordionGroup>
  <Accordion icon="user-plus" title="Sign up for Benchwrk">
    1. Visit [benchwrk.com](https://benchwrk.com) and click "Sign Up"
    2. Enter your email and create a password
    3. Verify your email address
  </Accordion>
</AccordionGroup>

### Step 2: Subscribe to a plan

<AccordionGroup>
  <Accordion icon="credit-card" title="Choose and subscribe to a plan">
    1. After logging in, go to the Billing section in your dashboard
    2. Select a plan that fits your needs (Free, Pro, or Enterprise)
    3. Enter your payment information if required
    4. Confirm your subscription
  </Accordion>
</AccordionGroup>

### Step 3: Add your sources

<AccordionGroup>
  <Accordion icon="plus-circle" title="Configure your logging sources">
    1. In your Benchwrk dashboard, navigate to "Sources"
    2. Click "Add Source" to create a new logging source
    3. Give your source a name (e.g., "My Web App")
    4. Configure the source settings as needed
  </Accordion>
</AccordionGroup>

### Step 4: Create your API key

<AccordionGroup>
  <Accordion icon="key" title="Generate API key from settings">
    1. Go to Settings in your Benchwrk dashboard
    2. Navigate to the "API Keys" section
    3. Click "Create New API Key"
    4. Give your key a name and select the appropriate permissions
    5. Copy the generated API key (keep it secure!)
  </Accordion>
</AccordionGroup>

### Step 5: Set up the Chrome extension

<Accordion icon="puzzle-piece" title="Install and configure Benchwrk Chrome extension">
  1. Install the Benchwrk [Chrome extension](https://chromewebstore.google.com/detail/benchwrk/bbipbannkhfdbiedliabpdiglcboinan) from the Chrome Web Store
  2. Open the extension and click on settings
  3. Paste your API key from Step 4 into the API key field
  4. Select your source from the dropdown
  5. Save the settings

  <Tip>Your extension is now ready to receive logs from your applications!</Tip>
</Accordion>

### Step 6: Set up vscode extension

  1. Install the Benchwrk [VSCode extension](https://marketplace.visualstudio.com/items?itemName=Benchwrk.benchwrk)
  2. Open the extension panel
  3. Paste your API key from Step 4 into the API key field
  4. Save the settings
